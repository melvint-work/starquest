# STARQUEST General Onboarding

⭐ Welcome to the **STARQUEST General Onboarding**!

STARQUEST is the official onboarding expedition designed to transform newcomers into mission-ready contributors at STARLAB. Rather than sitting through endless lectures, this repository contains a series of interactive, hands-on scenarios designed to test and improve your knowledge on the tools and operational best practices.

These challenges are hosted on [Killercoda](https://killercoda.com/), providing an interactive, browser-based environment where you can deploy manifests, debug issues, and secure clusters in real-time.
